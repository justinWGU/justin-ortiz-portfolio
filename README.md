# Personal Website & Project Showcase

## Overview

A personal website used to present selected software engineering projects and provide a single place to review my work and background.

## What It Does

- Showcases projects involving full-stack web development, AI integration, and embedded systems
- Provides direct links to project repositories and demos
- Offers an easy way to access my resume and contact information

## Technical Notes

- Designed to be clean, minimal, and easy to scan for recruiters
- Focuses on clarity and engineering context over visual complexity

## Current Status

Live and actively updated as projects evolve.

## Built With

React, Tailwind CSS, Netlify
